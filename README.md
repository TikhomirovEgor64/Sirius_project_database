# Sirius_project_database
При открытии программы отображается окно, в котором нужно выбрать файл базы данных (с расширением db), который будет дан вместе с файлом exe. После корректного выбора пользователь может перейти в любое окно, которое есть в меню. (интерфейс представлен на рис. 1 в Приложении 1)

Описание меню: «База данных» («Импорт данных», «Открыть базу данных», «Импорт трафика», «Импорт начислений», «Импорт абон. номеров», «Параметры отображения», «Условия договора»), «Сводная таблица» («Открыть сводную таблицу по категориям», «Открыть сводную таблицу по тарифам», «Рассчитать затраты», «Прогнозный график»), «Удаление информации» («Удаление информации»), «Сохраненная информация» («Все сводные таблицы»).

«Импорт данных» (интерфейс представлен на рис. 2 в Приложении 1): открывается окно с двумя кнопками («Выбрать файл», «Импортировать»). При нажатии на кнопку «Выбрать файл» открывается диалоговое окно, в котором нужно выбрать файл Excel (с расширением xls), из которого будут импортированы данные, имеющиеся на данный момент. После выбора на окне появится текст с указанием полного пути к выбранному файлу. При нажатии на кнопку «Импортировать» данные из выбранного файла сохранятся в базе данных.

«Открыть базу данных» (интерфейс представлен на рис. 3 в Приложении 1): открывается окно с таблицой, в которой отображаются данные из базы данных. Данные можно редактировать, удалять, добавлять. Для удаления нужно выбрать строчки, которые нужно удалить, а затем нажать кнопку «Удалить».

«Импорт трафика» (или «Импорт начислений» или «Импорт абон. номеров») (интерфейс представлен на рис. 4-6 в Приложении 1): открывается окно с двумя кнопками («Выбрать файл», «Импортировать»), также на окне есть кнопки, с помощью которых можно выбрать месяц, за которые импортируются данные. При нажатии на кнопку «Выбрать файл» открывается диалоговое окно, в котором нужно выбрать файл Excel (с расширением xls), из которого будут импортированы данные о трафике (или начислениях или абонентских номерах). После выбора на окне появится текст с указанием полного пути к выбранному файлу. При нажатии на кнопку «Импортировать» данные из выбранного файла сохранятся в базе данных за выбранный месяц.

«Параметры отображения» (интерфейс представлен на рис. 7 в Приложении 1): открывается окно с кнопками, с помощью которых можно выбрать месяца, за которые нужно отображать данные. Также на окне есть кнопка «Сохранить изменения», после нажатия на которую данные будут иззменены.

«Условия договора» (интерфейс представлен на рис. 8 в Приложении 1): открывается окно, в котором можно внести условия договора (год начала действия договора, месяц начала действия договора, срок действия договора (в месяцах), количество тарифов по данному договору, денежная сумма по договору). Также есть таблица тарифов, в которую пользователь должен внести номера тарифов и их описание. После заполнения параметров договора и информации о тарифах пользователю нужно нажать кнопку «Сохранить» для сохранения условий договора.

«Открыть сводную таблицу по категориям» (или «Открыть сводную таблицу по тарифам») (интерфейс представлен на рис. 9-10 в Приложении 1): открывается окно, отображается сводная таблица по категориям (или тарифам).

«Рассчитать затраты» (интерфейс представлен на рис. 11 в Приложении 1): открывается окно, на котором пользователь может выбрать месяца и год, за которые нужно рассчитать затраты. На окне есть две кнопки («по категориям», «по тарифам»). При нажатии на кнопку «по категориям» (или «по тарифам») будет произведен расчет затрат за выбранные периоды, и полученные результаты будут добавлены в сводную таблицу по категориям (или по тарифам).

«Прогнозный график» (интерфейс представлен на рис. 12 в Приложении 1): открывается окно на котором отображается прогнозный график. На графике на горизонтальной оси отложены месяца из срока договора, на горизонтальной — затраченная денежная сумма. На конечном промежутке график будет выглядеть, как горизонтальная прямая. Значение ординаты этой прямой будет является усредненной денежной суммой, которую можно тратить в месяц на оставшемся периоде договора. Если на графике есть отрицательное значение денежной суммы, это значит, что денежная сумма выделенная по договору закончилась.

«Удаление информации» (интерфейс представлен на рис. 13 в Приложении 1) открывается окно с тремя кнопками («Удалить информацию по месяцам из базы данных», «Удалить сводную таблицу (без сохранения)», «Удалить сводную таблицу (с сохранением)»). При нажатии на кнопку «Удалить информацию по месяцам из базы данных» данные по месяцам удалятся из базы данных. При нажатии на кнопку «Удалить сводную таблицу (без сохранения)» сводные таблицы удалятся без сохранения. При нажатии кнопку «Удалить сводную таблицу (с сохранением)» сводные таблицы удалятся, но при этом сохранятся, и они будут доступны («Сохраненная информация» → «Все сводные таблицы»), также сводные таблицы экспортируются в файл Excel (расширение xls) в нужную папку на компьютере (при нажатии на кнопку будет открыто диалоговое окно, с помощью которого можно указать путь к сохраненному файлу).

«Все сводные таблицы» (интерфейс представлен на рис. 14 в Приложении 1): открывается окно с таблицой, в которую занесены данные о сохраненных сводных таблицах. Если нажать на одну из строчек таблицы, то будет открыто окно с соответствующей сводной таблицей.

В случае отстутствия каких-либо данных на окне появится соответствующая текстовая информация. В случае какой-либо ошибки при работе с программой появится диалоговое окно с подсказкой, как исправить ошибку. В случае успешного завершения какого-либо процесса появится диалоговое окно с соответствующей информацией. Чтобы закрыть диалоговое окно нужно нажать кнопку «close» на диалоговом окне.
